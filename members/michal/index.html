NI-CCC Project

My project proposal is to visualise time distances between places when using public transport. This seems trivial, but could help a lot. To make it a little more interesting, I've come up with the following ideas:

  • Strictly use only open data. Strive to use the most universal formats possible.
  • Make it modular. Visualising one city or another, or even a country or a larger region should really be the same if the data is standardized.
  • Do some research and come up with fun metrics. Such as, what is the time distance to all other places in Prague, but at 3:15 in the morning. Or how far you can get with 120 Kc or something.
  • Make an editor, so that you can visualise what effect your new metro line would have if you route it as you intend it to way. Or so that you can visualise what effect would it have if a particular track would go out of order.
  • Use real time data. This is scary, though. But interesting. But scary.
