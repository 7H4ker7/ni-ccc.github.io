Crime Map

Projekt předmětu NI-CCC v zimním semestru 2021-2022

Autoři: Duong Tat Dat, Viktor Káčer

Nápad

Nedávné době jsme si s přítelkyní hledali byt a zajímalo nás, jak je na tom Praha co se týče kriminality. Zda platí přísloví “Žižkovu a Libni, zdaleka se vyhni...”

Hlavní myšlenkou je vytvořit pohled ptačím okem nad městem (Prahou), kde by pozorovatel mohl nabýt dojmu, jak se odvíjí kriminalita ve městě v čase. Index kriminality / zabarvení celého kraje je čistý statistický údaj a chtěli bychom vizualizovat, co toto znamená.

Plán

  • Domluvit se na vizuálním provedení
  • Vykreslování mapy na webu
  • Vykreslování modelu postav na webu
  • Příprava modelu
  • Zpracování dat a tahání skrz API

Inspirace

Platforma

  • Webová aplikace díky přístupnosti
  • Načítání dat z OpenStreetMaps / Mapbox
